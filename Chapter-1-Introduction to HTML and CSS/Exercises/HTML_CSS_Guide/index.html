HTML & CSS Quick-Reference Guide

A simple guide to help you remember essential HTML tags and CSS properties.

HTML Tags

<h1> to <h6>

Used to define headings, from largest (<h1>) to smallest (<h6>).

<h1>Main Heading</h1>

<p>

Defines a paragraph of text.

<p>This is a paragraph.</p>

<a>

Creates a hyperlink to another page or website.

<a href="https://example.com">Visit</a>

<img>

Embeds an image.

<img src="image.jpg" alt="Description">

<ul> & <li>

Defines an unordered list and list items.

<ul>
                    <li>Item 1</li>
                    <li>Item 2</li>
                    </ul>

<ol> & <li>

Defines an ordered (numbered) list.

<ol>
                    <li>First</li>
                    <li>Second</li>
                    </ol>

<div>

A container element used to group content for styling or scripting.

<div>Content here</div>

<span>

Inline container for text that can be styled.

<span style="color: red;">Red text</span>

<strong>

Makes text bold and emphasizes it.

<strong>Important!</strong>

<br>

Inserts a line break.

Line 1<br>Line 2

CSS Properties

color

Sets the text color.

color: #333;

font-size

Changes the size of the text.

font-size: 16px;

background-color

Sets the background color of an element.

background-color: #f5f5f5;

padding

Adds space inside the element’s border.

padding: 20px;

margin

Adds space outside the element’s border.

margin: 10px;

border

Adds a border around the element.

border: 1px solid #ccc;

text-align

Aligns text inside a block element.

text-align: center;

font-family

Sets the font of the text.

font-family: Arial, sans-serif;
